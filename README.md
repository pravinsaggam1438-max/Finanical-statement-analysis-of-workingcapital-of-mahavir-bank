# 📊 Study on Analysis of Financial Statement
## Shri Mahavir Urban Co-op Bank Ltd., Solapur
---

## 🎯 Objective
To analyze the financial performance of Shri Mahavir Urban 
Co-op Bank Ltd., Solapur using ratio analysis, trend analysis, 
and comparative statements to evaluate overall financial health.

## ❓ Problem Statement
Co-operative banks often face challenges in maintaining 
financial transparency and performance tracking. This study 
aims to evaluate the financial position of Mahavir Urban 
Co-op Bank using structured financial analysis tools.

## 🛠️ Tools Used
- Microsoft Office (Excel, Word)
- Financial Analysis Tools
- Ratio Analysis
- Trend Analysis
- Comparative Statement Analysis

## 📊 Analysis Performed
| Analysis Type | Details |
|--------------|---------|
| Ratio Analysis | Liquidity, Profitability, Solvency Ratios |
| Trend Analysis | Year-wise performance tracking |
| Comparative Statements | Year on year comparison |
| Financial Position | Strengths and weaknesses identified |

## 📈 Key Findings
- ✅ Conducted financial analysis using ratio analysis, 
  trend analysis and comparative statements
- ✅ Evaluated bank performance and financial position
- ✅ Calculated financial ratios to identify strengths 
  and weaknesses
- ✅ Interpreted results to provide meaningful insights 
  on bank's financial health

## 💡 What I Learned
- Real-world financial statement analysis
- Practical application of ratio and trend analysis
- Co-operative banking operations and structure
- How to interpret financial data meaningfully
